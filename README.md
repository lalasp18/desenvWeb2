# Sobre o projeto

O **desenvWeb2** é um sistema web desenvolvido na disciplina **Desenvolvimento Web II**, com o objetivo de aplicar na prática conceitos de desenvolvimento full stack, integrando lógica de back-end em Java com interface web.

O projeto foi inicialmente desenvolvido no **Apache NetBeans** e posteriormente evoluído em versões que exploram diferentes abordagens de front-end e back-end, incluindo uma implementação com **Angular + Java**.

---

## Objetivo

O objetivo do projeto é consolidar conhecimentos de desenvolvimento web, com foco em:

- Estruturação de aplicações web
- Implementação de regras de negócio no back-end
- Organização de camadas da aplicação
- Evolução de uma aplicação acadêmica para diferentes arquiteturas

Este trabalho foi desenvolvido para a disciplina **DesWeb II – Desenvolvimento Web II**, do curso de **Sistemas de Informação (SI)**.

---

## Tecnologias utilizadas

- Java 11
- JSP / Servlets
- HTML
- CSS
- Apache Tomcat 9

---

## Estrutura do projeto

- Back-end em Java responsável pela lógica de negócio
- Camada web com JSP ou interface separada (dependendo da versão)
- Evolução para arquitetura front-end + back-end desacoplados na versão Angular + Java
- Organização voltada para prática acadêmica de sistemas web
